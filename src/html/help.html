Помощь

Компоненты:
  • Факультет - выбор доступных факультетов для отображения расписания
  • Обновить - принулительно обновляет таблицу расписаний
  • Кнопки раздела Таблицы - открывают окно редактирования данных
  • Кнопки раздела Отчёты - открывают окно просмотра отчётов
  • Помощь - окно руководства пользователя (вы как раз его читаете)
  • О программе - информация о разработчике приложения
  • Выход - закрывает окно приложения и освобождает ресурсы.
